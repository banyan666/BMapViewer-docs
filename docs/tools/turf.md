# Turf 高级地理空间分析工具库
BMapViewer内置了turf地理空间分析工具库，方便开发者进行地理空间分析。

## 使用示例

```js
import { turf } from "@/components/BMapViewer/b-map-viewer.js";

turf.point([116.400219, 39.898231])


```
